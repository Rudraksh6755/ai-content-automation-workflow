# 🚀 AI Content Automation Workflow using n8n

Turn articles into AI-generated social media posts automatically using workflow automation.

## 🧠 System Architecture

Article URL
   ↓
Content Extraction
   ↓
AI Summarization (LLM)
   ↓
Post Generation
   ↓
Automated Publishing via n8n

## 🎯 Why this project?

This project demonstrates how AI + workflow automation can eliminate repetitive manual tasks and build scalable content pipelines.
